## 🚀 Projects

### GLAM-ID System
**Client:** FG Aesthetic Centre  
A full-stack, web-based customer information and loyalty management system. 
*   **NFC Loyalty Integration:** Utilizes USB HID keyboard-emulation readers for client check-in, automatic point accrual, and point redemption at checkout.
*   **Customer & Appointment Management:** Centralized database for comprehensive client profiles including health data, alongside scheduling for one-time and recurring treatments.
*   **Point of Sale & Inventory:** Processes transactions, generates Z-readings, and features automated inventory deduction on sales.
*   **Role-Based Security:** Implements three access tiers (Super Admin, Branch Admin, Staff) with data isolation strictly enforced via Supabase Row Level Security (RLS).
*   **Tech Stack:** React 19, TypeScript, Tailwind CSS 4, Shadcn/UI, Flask 3.0.0, and Supabase.
